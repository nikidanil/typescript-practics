# Задача 2: Нормализация данных (`normalize-data`)

## Описание задачи

Преобразовать массив объектов постов в нормализованную структуру. Нормализация — это техника организации данных, при которой они хранятся в виде словаря (объекта), где ключ — это ID элемента, а значение — сам элемент. Это позволяет получать элемент по ID за время O(1).

## Решаемая проблема / Изучаемая концепция

**Проблема:** Поиск элемента по ID в массиве требует итерации (O(n)), что неэффективно при частых операциях выборки.

**Решение в TypeScript:** Корректное описание структуры нормализованных данных с использованием **индексированных свойств** (`[id: string]: Post`). Это позволяет TypeScript проверять типы значений, доступных по любому строковому ключу внутри объекта `byId`.

## Ключевые элементы решения

1.  **Интерфейс `NormalizedData`:**
    *   `byId`: Объект, где ключом является строка (ID поста), а значением — объект типа `Post`. Это **индексированный тип**.
    *   `allIds`: Массив ID всех постов, сохраняющий исходный порядок.
2.  **Функция `normalizeData`:** Итерируется по входному массиву, заполняя объект `byId` и массив `allIds`.

## Как запустить и протестировать

1.  Убедитесь, что вы находитесь в ветке `normalize-data`:
    ```bash
    git checkout normalize-data
    ```
2.  Запустите код:
    ```bash
    npm run dev
    ```
    Вы должны увидеть в консоли объект с двумя полями: `byId` (содержащий словарь постов) и `allIds` (содержащий массив ID).

3.  **Для проверки типов:** Попробуйте обратиться к несуществующему ID в объекте `result.byId`. TypeScript не выдаст ошибку на этапе компиляции (так как ключ динамический), но тип значения будет известен — `Post`.

## Пример использования

```typescript
// Исходные данные
const posts = [
  { id: '1', title: 'Title 1', body: 'Body 1' },
  { id: '2', title: 'Title 2', body: 'Body 2' },
];

// Нормализованные данные
const normalizedData = normalizeData(posts);
/*
{
  byId: {
    '1': { id: '1', title: 'Title 1', body: 'Body 1' },
    '2': { id: '2', title: 'Title 2', body: 'Body 2' }
  },
  allIds: ['1', '2']
}
*/

// Быстрый доступ к посту по ID
const post1 = normalizedData.byId['1']; // Тип `Post`
